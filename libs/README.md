# libs/

NHN Cloud Log & Crash Search SDK JAR 파일을 이 디렉토리에 배치합니다.

## 필요한 파일

- `logncrash-java-sdk3-4.0.0.jar`

## 다운로드 방법

1. [NHN Cloud 콘솔](https://console.nhncloud.com) 접속
2. Data & Analytics > Log & Crash Search 선택
3. 설정 > SDK 다운로드에서 **Logback SDK** 다운로드
4. 압축 해제 후 `logncrash-java-sdk3-4.0.0.jar`를 이 디렉토리에 복사

## 참고

- 이 SDK는 Maven Central에 배포되지 않아 수동 다운로드가 필요합니다.
- JAR 파일은 `.gitignore`에 의해 Git에 커밋되지 않습니다.
- Docker 빌드 시 이 디렉토리가 빌드 컨텍스트에 포함됩니다.
